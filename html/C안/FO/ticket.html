수험표 출력 안내

시험일로부터 2주 전, 월요일 오전 10시부터 TOPIK 홈페이지에서 수험표 출력이 가능합니다. (월요일이 공휴일인 경우에는 다음 날 오전 10시)

수험표 확인 방법

  1. TOPIK 홈페이지(www.topik.go.kr) 접속
  2. 회원 로그인 → [마이페이지] → [접수현황] → [수험표 출력]
  3. 비회원 또는 ID가 기억나지 않거나 탈퇴한 경우, [전체메뉴] → [비회원 시험정보] → [수험표 출력]

※ 응시불가 처리된 지원자는 수험표 출력 불가

시험 당일 안내

수험표를 출력하여 시험 당일 규정 신분증과 함께 지참하셔서 시험장을 방문하시기 바랍니다.

부득이 수험표를 출력하지 못하는 경우에는 사진으로 캡쳐해서 가지고 가시면 본인의 시험실 및 자리를 쉽게 찾으실 수 있습니다.

수험표 출력 시 유의사항

  1. 한국어능력시험 수험표 출력 시 Chrome 브라우저 사용을 권장합니다.
  2. 학교·도서관·센터 내 공용 인터넷 검색용 PC에서는 증명서 출력 프로그램이 설치되지 않거나 정상 작동하지 않을 수 있으니, 개인 PC 또는 출력 가능한 기타 장소에서 진행해 주시기 바랍니다.
  3. 출력 버튼을 눌러도 화면이 나타나지 않는 경우, 해당 브라우저의 설정에서 ‘팝업 차단’이 되어 있는지 확인하시고 해제 후 다시 시도해 주시기 바랍니다.
