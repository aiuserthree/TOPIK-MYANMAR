접수 방법

회원가입부터 수험표 출력까지, TOPIK 시험 접수의 전체 흐름을 안내합니다.

접수 방법 시험 접수 접수 확인 수험표 출력

접수 절차

1

회원가입

이메일 인증 후 기본정보·증명사진을 등록합니다. 등록된 사진은 시험 접수 시 자동으로 사용됩니다.

온라인
2

로그인

회원가입 시 인증한 이메일과 비밀번호로 로그인합니다. Google 계정으로도 가능합니다.

온라인
3

시험 접수 (4단계)

회차·급수 선택 → 회원정보 확인 → 사진 규격 확인 → 최종 제출. TOPIK Ⅰ과 Ⅱ는 동시 접수 가능합니다.

온라인
4

사진 심사 + 오프라인 응시료 수납

제출된 사진은 운영진의 사진 심사를 거치며, 응시료는 2026.07.24(금) ~ 07.26(일)에 지정된 수납처에서 직접 납부합니다. 동시 접수 시 각 수준별로 개별 납부합니다.

오프라인
5

수험번호 부여

사진 승인과 수납 완료가 모두 확인되면 일괄적으로 수험번호가 부여됩니다. 마이페이지 · 접수 확인에서 확인할 수 있습니다.

온라인
6

수험표 출력

시험일 2주 전부터 TOPIK 본부(topik.go.kr)에서 수험표를 출력합니다. 시험 당일 신분증과 함께 지참해 주세요.

외부 사이트

준비물 체크리스트

  • 유효한 이메일 주소 (인증코드 수신용)
  • 본인 명의 연락처 (미얀마 휴대전화 권장)
  • 여권용 증명사진 (JPG·PNG, 3:4 비율, 200KB~2MB, 6개월 이내)
  • 여권 또는 NRC 등 신분증 정보 (영문 성명·생년월일 확인용)
  • 응시료 (오프라인 수납 시 필요)
본 사이트에서는 응시료를 온라인으로 결제할 수 없습니다. 접수 완료 후 지정된 오프라인 수납처에서 직접 납부해 주세요. 수납 완료 후 수험번호가 부여됩니다.
