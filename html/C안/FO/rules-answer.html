사용 가능한 필기구

검정색 컴퓨터용 사인펜만 사용 가능합니다. 시험장에서 제공되며, 응시자 본인 휴대도 가능합니다. 수정테이프·수정액·연필·볼펜은 사용 금지입니다.

OMR 카드 마킹 방법

1. ① ② ③ ④

올바른 예 — 원 안을 완전히 채워 진하게 마킹

2. ① ② ③ ④

잘못된 예 — 빗금·점·체크 표시는 인식되지 않습니다

해야 하는 것

  • 원의 안쪽을 완전히 채워 진하게 마킹
  • 검정색 컴퓨터용 사인펜 사용
  • 한 문항당 한 개의 답만 마킹
  • 응시자 코드·수험번호 정확히 마킹

하지 말아야 할 것

  • 빗금(/), 점(.), 체크(✓) 표시 사용
  • 수정테이프·수정액 사용
  • 한 문항에 두 개 이상 마킹 (오답 처리)
  • OMR 카드 접거나 구기기

쓰기 답안지 작성 (TOPIK Ⅱ)

쓰기 영역은 별도 답안지에 직접 손글씨로 작성합니다. 200~300자 단문(2문항)과 600~700자 작문(2문항)으로 구성되며, 한 칸에 한 글자씩 원고지 규칙에 따라 작성해 주세요.
· 문장 부호(., ?, !)는 한 칸을 차지합니다.
· 문단을 바꿀 때 첫 칸은 비우고 두 번째 칸부터 작성합니다.
· 알파벳·숫자는 한 칸에 두 글자씩 작성합니다.
· 글자 수가 부족하거나 초과해도 채점됩니다. 다만 권장 분량 유지를 권합니다.

응시자 코드·수험번호 기재

· 수험표에 기재된 수험번호 8자리를 OMR 상단 해당 칸에 정확히 마킹합니다.
· 응시자 코드는 시험장에서 별도 안내되는 코드를 마킹합니다.
· 잘못 마킹된 코드·번호로 인한 채점 오류는 응시자 본인 책임입니다.
