| 주요 검증 사항 | ### 1. 데이터 정합성 |
|  | - 모든 통계는 동일 데이터 소스(서버 집계) 사용 |
|  | - 통계 카드 합계와 최근 접수 현황 일치 |
|  | - 상태별 카운트 = 접수자 목록의 필터 카운트와 일치 |
|  | ### 2. 활성 회차 선택 |
|  | - 대시보드는 기본적으로 활성 회차 데이터 표시 |
|  | - 회차 전환 셀렉트(향후) 추가 시, 모든 카드/테이블 동기화 |
|  | ### 3. 갱신 정책 |
|  | - 페이지 진입 시 즉시 로딩 (loading 인디케이터) |
|  | - 자동 갱신 시 페이지 깜빡임 최소화 (in-place 업데이트) |
|  | - 갱신 시각 표시 (예: "10초 전 업데이트") |
|  | ### 4. 접근성 |
|  | - 통계 카드: 라벨 + 값을 시각적/음성 안내 프로그램 모두 명확 |
|  | - 테이블: 헤더 + 정렬 가능한 컬럼 표시 |
|  | - 색상만으로 정보 전달 금지 (배지에 텍스트 병행) |
|  | ### 5. 모바일 |
|  | - 통계 카드: 1열 또는 2열 wrap |
|  | - 테이블: 가로 스크롤 |
|  | - 폰트 크기 축소 |
