TOPIK Ⅰ과 Ⅱ를 동시에 접수할 수 있나요?
네, 동일 회차 내에서 TOPIK Ⅰ과 TOPIK Ⅱ를 동시에 접수하실 수 있습니다. 두 시험은 동일한 시험장 코드로 부여되며, 응시료는 수준별로 각각 오프라인 수납됩니다.
응시료는 어떻게 납부하나요?
응시료는 온라인 결제가 아닌 오프라인 수납으로만 가능합니다. 공지사항에 안내된 수납처에서 직접 납부해 주시기 바랍니다. 수납이 확인된 후 수험번호가 부여됩니다.
접수 후 정보(이름·생년월일 등)를 잘못 입력했습니다.
연락처·이메일·비밀번호·사진은 마이페이지에서 직접 수정하실 수 있습니다. 성명·생년월일·국적 등 본인 식별 정보 정정은 환불·정보정정신청 게시판을 통해 신청해 주세요.
