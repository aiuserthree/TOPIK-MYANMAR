채점 방식

객관식은 OMR 자동 채점, 쓰기(주관식)는 평가위원의 채점을 거칩니다. 영역별 점수가 모두 산정되어 등급이 결정됩니다.

성적 인정 기간

TOPIK 성적은 발표일로부터 2년간 유효합니다. 유학·취업·이민 등 활용 시 만료일을 확인해 주세요.

등급 부여 방식

TOPIK Ⅰ·Ⅱ 모두 총점 기준으로 자동 등급이 부여됩니다. 별도 합격·불합격 구분은 없으며, 등급 미달 시 ‘불합격’으로 표기됩니다.

성적표 발급

합격 발표 후 약 1주일 뒤부터 TOPIK 본부 홈페이지(topik.go.kr)에서 PDF 성적표를 발급받으실 수 있습니다.
