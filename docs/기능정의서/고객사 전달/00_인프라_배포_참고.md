**작성일:** 2026-06-06  
**대상:** 본 폴더의 FO/BO **기능정의서(xlsx)** 는 화면·업무 요구사항만 기술합니다. 아래는 **현재 개발·운영 인프라** 요약이며, 기능 범위와 무관한 기술 스택 변경 시 본 문서만 갱신합니다.

## 1. 운영 목표 (확정)

| 계층 | 기술 | 비고 |
| --- | --- | --- |
| FO | `https://www.topik-myanmar.com` | IwinV Web VPS — nginx 정적 (`apps/web/dist/` 중기, 단기 `build.py` FO) |

## 2. 이전 dev/UAT 스택 (부록 — 운영 아님)

과거 프로토타입·임시 연동에 사용했던 구성입니다. **고객사 DNS·운영 합의의 기준이 아닙니다.**

| 항목 | 임시 URL/서비스 |
| --- | --- |
| FO | Vercel (`topik-myanmar.vercel.app`) |
| API | Railway (`topikmyanmar-production.up.railway.app`) |
| 메일 | Resend API (`MAIL_PROVIDER=resend`) — **운영 대안**으로만 문서에 유지 |

운영 전환 시 FO/API/메일/DNS는 §1 및 [`docs/고객사_DNS_요청_템플릿.md`](../../고객사_DNS_요청_템플릿.md)를 따릅니다.

---

## 3. API·메일 구현 현황 (2026-06-06)

- **백엔드:** `apps/api` (Python FastAPI) — FO `/api/v1/*`, BO `/api/v1/admin/*` 대부분 구현
- **DB:** `db/migrations/V001`~`V005` SQL migration 저장소 포함
- **이메일:** `email_outbox` + SMTP/Resend/console (`apps/api/app/lib/mail.py`)
- **HTML 템플릿:** C안 에디토리얼 — `시안/email/` (미리보기), API 렌더 `apps/api/app/lib/email_render.py` (가입 인증·비밀번호 재설정 등)
- **SMTP 스모크:** `python3 scripts/test_smtp.py --to <메일>`

## 4. 기능정의서와의 관계

| 문서 | 내용 |
| --- | --- |
| `FO/*.xlsx`, `BO/*.xlsx` | 화면 IA·입력 검증·업무 흐름 (인프라 비포함) |
| `정책_합의_워크시트.md` (상위 폴더) | 도메인·메일·OAuth 등 **운영 합의** — §1 값과 동기화 |
| 본 문서 | 기술 스택·배포 경로만 기술 |

기능정의서 본문의 「SMTP·SES 등 운영 합의」 표현은 **발송 채널 일반**을 뜻하며, 현재 구현 기본값은 **IwinV 테라웹메일 SMTP**입니다.
