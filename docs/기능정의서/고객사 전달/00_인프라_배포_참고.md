# 인프라·배포 참고 (고객사 전달용)

**작성일:** 2026-06-06 · **갱신:** 2026-06-09  
**대상:** 본 폴더의 FO/BO **기능정의서(xlsx)** 는 화면·업무 요구사항만 기술합니다. 아래는 **현재 개발·운영 인프라** 요약이며, 기능 범위와 무관한 기술 스택 변경 시 본 문서만 갱신합니다.

---

## 1. 운영 목표 (확정)

| 계층 | 기술 | 비고 |
| --- | --- | --- |
| FO | `https://www.topik-myanmar.com` | IwinV Web VPS — nginx 정적 (`build.py` → `public/`. `apps/web`은 중기) |
| BO | `https://admin.topik-myanmar.com` | 동일 Web VPS — handoff `html/C안/BO(admin)/project/` 또는 `/admin/` |
| API | FastAPI (`apps/api`) | 동일 Web VPS — systemd `myanmar-api`, nginx `/api/` 프록시 |
| DB | PostgreSQL 15+ | IwinV DB VPS (`115.68.227.1`) |
| 파일 | IwinV 오브젝트 스토리지 (S3 호환) | `https://kr.object.iwinv.kr`, Private 버킷 + API 프록시 |
| 메일 | IwinV **테라웹메일 SMTP** | `MAIL_PROVIDER=smtp`, `noreply@topik-myanmar.com` |

- 상세 절차: [`docs/IWINV_SETUP.md`](../../IWINV_SETUP.md), 개발 스펙: [`docs/DEV_SPEC.md`](../../DEV_SPEC.md)

---

## 2. API·메일 구현 현황 (2026-06-09)

- **백엔드:** `apps/api` (Python FastAPI) — FO/BO REST **1단계 구현 완료**
- **FO/BO 화면:** `html/C안/FO/` (25페이지), `html/C안/BO(admin)/project/` (패널 13개)
- **DB:** `db/migrations/V001`~`V008` (pgvector, `exam_venues.name_my` 포함)
- **이메일:** `email_outbox` 워커 + C안 14종 ko/my/en 렌더
- **Google OAuth:** API 구현 완료 — `GOOGLE_CLIENT_ID` 설정 시 가동
- **SMTP 스모크:** `python3 scripts/test_smtp.py --to <메일>`

## 3. 기능정의서와의 관계

| 문서 | 내용 |
| --- | --- |
| `FO/*.xlsx`, `BO/*.xlsx` | 화면 IA·입력 검증·업무 흐름 (인프라 비포함) |
| `정책_합의_워크시트.md` (상위 폴더) | 도메인·메일·OAuth 등 **운영 합의** — §1 값과 동기화 |
| 본 문서 | 기술 스택·배포 경로만 기술 |

기능정의서 본문의 「SMTP·SES 등 운영 합의」 표현은 **발송 채널 일반**을 뜻하며, 현재 구현 기본값은 **IwinV 테라웹메일 SMTP**입니다.
