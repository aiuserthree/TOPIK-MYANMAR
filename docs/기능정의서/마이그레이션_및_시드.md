# 마이그레이션 및 시드

**작성일:** 2026-06-02 · **갱신:** 2026-06-09  
**상태:** V001~V008 적용 · FastAPI(`apps/api`) 운영 기준

---

## 1. 마이그레이션 도구

| 항목 | 결정 |
|------|------|
| **정본** | 순차 SQL `db/migrations/V{nnn}__description.sql` |
| **적용** | `bash scripts/run-migrations.sh` (V001~V008 일괄) |
| **DB** | PostgreSQL 15+ + pgvector |
| **Alembic** | `apps/api/alembic/` — 신규 빈 DB bootstrap용 1 revision만. **운영은 SQL 기준** |
| **레거시** | 초안의 Flyway/Node Drizzle — **미사용** |

### 버전 관리 규칙

| 규칙 | 내용 |
|------|------|
| 파일명 | `V001__initial_schema.sql` … `V008__exam_venue_name_my.sql` — **적용된 파일 수정 금지** |
| 롤백 | forward migration만 — 수정은 새 `V00n` 추가 |
| V007 | `CREATE EXTENSION vector` — **postgres superuser** + stdin (`< 경로`) |
| 시드 | Flyway/SQL 외부 — `scripts/seed_dev.py`, `scripts/seed_prod.py`, `scripts/create_admin.py` |

## 4. 실행

```bash
# 마이그레이션
createdb topik_myanmar
bash scripts/run-migrations.sh
sudo -u postgres psql -d topik_myanmar < db/migrations/V007__pgvector_semantic_search.sql

# 개발 시드
python3 scripts/seed_dev.py

# 운영
CONFIRM_PROD_SEED=1 python3 scripts/seed_prod.py
ADMIN_EMAIL=... ADMIN_PASSWORD=... python3 scripts/create_admin.py
```
