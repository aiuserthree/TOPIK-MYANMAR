## 1. 서비스 개요

- **목적**: 한국어능력시험(TOPIK)의 정의·목적·주관/시행 기관·등급 구분·문항 구성·평가 기준을 4개 하위 페이지로 안내한다. (0519 재구성: 기존 단일 '안내' 페이지 → 4분할)
- **범위**: 정적 콘텐츠 4페이지. **데이터 쓰기 없음**. 본문은 다국어 키(i18n) 기반으로 운영자가 번역만 갱신. 0527로 시험 개요 페이지 상단에 외부 링크 버튼 3종 추가.
- **주요 액터**: 비로그인+로그인 전 사용자(읽기 전용).
- **관련 요구사항ID**: `TPKM_FO_REQ_016`, `TPKM_FO_REQ_017`, `TPKM_FO_REQ_002`(다국어)

## 3. 핵심 비즈니스 규칙

- **콘텐츠 운영 모델**: 정적 페이지지만 i18n 키 기반으로 운영자가 번역만 갱신. KO 마스터 → MY/EN 번역(자동 번역 금지, 운영자 검수 후 게시).
- **다국어 키 정합**: `info.overview` / `info.intro` / `info.questions` / `info.evaluation` 키가 KO·MY·EN 사전에 1:1 존재해야 하며, 누락 시 폴백(MY→EN→KO).
- **BO 연동 없음(현행)**: 본문은 정적/i18n. 향후 BO에 '안내 페이지 본문 관리' 추가 시 출력 대상이 됨(IA v1.1, 미착수).
- **데이터 흐름**: 본 서비스는 **DB 쓰기/상태 전이 없음**. 회차 강조용으로 `exam_rounds` 읽기만 선택적으로 사용.
