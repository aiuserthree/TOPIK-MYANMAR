이 가이드 사용법

  • 필수 표기: 입력 필드 표에서 ● = 반드시 입력, ○ = 선택 입력입니다.
  • 로그인이 필요한 메뉴: 시험 접수, 접수 확인, 환불·정정, 문의게시판은 로그인 후 이용할 수 있습니다. 로그인하지 않은 상태에서 해당 메뉴를 누르면 로그인 화면으로 이동하며, 로그인을 마치면 원래 가려던 페이지로 자동 이동합니다.
  • 언어 변경: 화면 오른쪽 상단의 KO / MY / EN 버튼을 누르면 한국어, 미얀마어, 영어로 전환됩니다. 선택한 언어는 다음 방문 때도 유지됩니다. 메뉴·버튼·입력 안내·오류 메시지(로그인 실패, OTP 오류, 접수 검증 등)도 함께 바뀝니다. 공지·FAQ·약관은 언어별로 내용이 다를 수 있으니, 편한 언어로 바꿔 읽어 보세요.

처음 이용하시는 분 — 추천 순서

  1. 메인 홈에서 현재 회차·접수 마감 확인
  2. 접수 방법과 응시료 규정 읽기
  3. 회원가입 → 로그인 → 시험 접수
  4. 접수 후 접수 확인(마이페이지)에서 사진 심사·수납·수험번호 진행 상황 확인
  5. 문제가 있으면 FAQ → 문의게시판 또는 환불·정보정정
TIP: 증명사진(JPG/PNG, 3:4, 200KB~2MB, 최근 6개월)과 오프라인 수납용 현금·접수번호를 미리 준비해 두면 접수가 훨씬 수월합니다. 이 사이트에서는 온라인 카드 결제가 되지 않습니다.
사용자 화면 (FO)01

공통 레이아웃 (GNB · 푸터 · 모바일)

모든 페이지는 같은 상단 메뉴(GNB), 하단 푸터, 모바일 하단 탭바를 사용합니다. 어느 화면에서든 메뉴 구성과 로그인 상태가 동일하게 보이므로, 한 번 익혀 두면 다른 페이지에서도 길을 잃지 않습니다.

사용 흐름 (시나리오)

  1. 페이지를 열면 상단 메뉴·하단 푸터·모바일 탭바가 자동으로 표시됩니다.
  2. 로그인한 상태라면 오른쪽에 내 이름과 로그아웃 버튼이 보이고, 로그인하지 않았다면 로그인·회원가입 버튼이 보입니다. 이름을 클릭하면 마이페이지(접수 확인) 로 이동합니다.
  3. 시험 접수·접수 확인·환불·정정·문의처럼 로그인이 필요한 메뉴를 누르면 먼저 로그인 화면으로 이동합니다. 로그인 후 원래 가려던 메뉴로 자동 이동합니다.
  4. 모바일에서는 화면 아래 4개 탭(홈·접수·게시판·마이)으로 주요 메뉴에 빠르게 갈 수 있습니다. 더 많은 메뉴는 우측 햄버거(≡) 를 눌러 주세요.

사용자 화면 (FO)02

메인 홈

사이트 첫 화면입니다. 지금 몇 회차가 접수 중인지, 마감까지 얼마나 남았는지, 시험·합격 발표 일정, 최신 공지, 자주 묻는 질문을 한곳에서 확인할 수 있습니다. 접수 전에 여기서 일정과 마감을 꼭 확인하세요.

요소별 상세

  • 1) 회차 배지 — 현재/다음 회차와 시험일을 표시합니다. 상태에 따라 모양이 바뀝니다: 접수 전 = "제N회 …(시험일)", 접수 중 = "… 접수 진행 중"(초록 점 깜빡임), 마감 = "접수 마감"(회색). 접수 중인 회차가 우선 표시되고, 없으면 가장 가까운 예정 회차가 표시됩니다.
  • 2) 주요 버튼 — 시험 접수하기(로그인 필요), 시험 안내 보기. PC는 가로, 모바일은 세로로 배치됩니다. 아직 가입하지 않았다면 회원가입 후 접수하세요.
  • 3) D-Day 카운트다운 — 접수 마감 시각까지 남은 일·시·분·초를 1초마다 갱신합니다. 0이 되면 "접수가 마감되었습니다 + 다음 회차 일정 보기" 안내로 자동 전환됩니다. 마감 직전에는 서버 부하로 지연될 수 있으니 여유 있게 접수하는 것을 권장합니다.
  • 4) 일정 타임라인 — 4단계 카드: 원서 접수 → 응시료 납부(오프라인) → 시험 시행 → 합격 발표(추후 공지). 현재 진행 중 단계는 진행중, 지난 단계는 종료 배지로 표시됩니다.
  • 5) 공지 미리보기 — 최신 공지 최대 5건. 각 행 = 카테고리 배지 + 제목(고정글은 NEW) + 날짜. 행을 클릭하면 공지 상세로 이동합니다. 중요·접수 공지는 반드시 읽어 보세요.
  • 6) 바로가기 카드 — 시험 접수 / 접수 확인 / 수험표 출력 / 환불·정보정정. 자주 쓰는 메뉴로 빠르게 이동합니다.
  • 7) FAQ 아코디언 — FAQ 상위 3건 요약. 질문 클릭 시 답변 펼침. 더 보려면 FAQ 메뉴로 이동하세요.
TIP: 홈의 D-Day는 접수 마감 기준입니다. 시험 당일까지 남은 시간과는 다릅니다. 접수 후에는 접수 확인(마이페이지)에서 사진 심사·수납 진행 상황을 확인하세요.

TOPIK 안내 · 시험 개요

TOPIK 시험의 전체 그림을 소개하는 페이지입니다. 시험이 무엇인지, 누가 볼 수 있는지, 미얀마에서 언제 시험이 열리는지 확인할 수 있습니다. 처음 TOPIK을 준비하시는 분은 이 페이지와 시험 소개를 함께 읽어 보세요.

번호요소무엇 / 동작
1외부 링크 3종주미얀마 대한민국 대사관·Facebook·YouTube (새 탭). 공식 소식·안내 영상을 볼 수 있습니다.
2핵심 요약 카드주관(국립국제교육원 NIIED) / 시행 기관(주미얀마 대한민국 대사관) / 응시 자격(국적 제한 없음).
3본문"TOPIK이란?"(1997년~ 시행 목적), 응시 목적 4가지(유학·취업·이민·자격 확인 등).
4연간 시험 일정 표등록된 시험 회차 일정이 표로 표시됩니다. 현재 접수 중인 회차는 표에서 강조 표시됩니다. 시각은 미얀마(UTC+6:30) 기준입니다.
5CTA 블록시험 접수하기 / 시험 규정 보기 / 자주 묻는 질문. 접수 전 규정·FAQ를 먼저 보면 실수를 줄일 수 있습니다.
TIP: 일정 표의 날짜는 변경될 수 있습니다. 최종 일정은 공지사항과 홈 화면 회차 정보를 함께 확인하세요.

TOPIK 안내 · 시험 소개

TOPIK Ⅰ과 Ⅱ의 차이, 등급별 활용처를 카드로 설명하는 안내 페이지입니다. "어느 급수를 접수해야 할까?" 고민될 때 참고하세요. Ⅰ·Ⅱ 동시 접수도 가능합니다(응시료는 급수별로 각각 납부).

번호요소내용
1TOPIK Ⅰ 카드초급(1·2급) · 2개 영역 · 100분 · 200점. 듣기 30 + 읽기 40(객관식). 급수별 어휘 안내. 한국어 기초~초급 학습자·EPS 등에 적합.
2TOPIK Ⅱ 카드중·고급(3~6급) · 3개 영역 · 180분 · 300점. 듣기 50 + 쓰기 4 + 읽기 50(객관식+주관식). 유학·전문직 등에 많이 사용.
3동시 접수 안내 콜아웃Ⅰ·Ⅱ 동시 접수 가능, 동일 시험장 코드, 응시료는 수준별 개별 오프라인 수납. 같은 날 두 시험을 보려면 접수 시 둘 다 선택하세요.
4활용처 카드 3종한국 유학(권장 3급+) / EPS·취업(권장 2급+) / 거주·이민(권장 3급+). 목적에 맞는 급수를 선택하는 참고 자료입니다.
TIP: 목표 기관(대학·고용주)에서 요구하는 급수를 먼저 확인한 뒤 접수하세요. 급수를 잘못 선택해도 접수 기간 중 수납 전까지는 마이페이지에서 취소할 수 있습니다.

TOPIK 접수 · 시험 접수 (4단계)

로그인 필수 — 이용 전 로그인해 주세요.

4단계로 나뉜 온라인 시험 접수 화면입니다. 상단 단계 표시에서 완료한 단계를 클릭하거나 다음·이전 버튼으로 이동합니다. 저장 후 나가기로 중간 저장(30일)도 가능합니다.

사용 흐름 (시나리오)

  1. STEP1 회차·급수 — 접수 중인 회차만 선택 → 시험장 → TOPIK Ⅰ/Ⅱ(복수 가능) → 다음.
  2. STEP2 회원정보 확인 — 가입 정보 읽기 전용. 수정은 내정보 수정에서 접수 전에.
  3. STEP3 사진 규격 확인 — 증명사진 + 5개 체크 전부해야 다음 활성. 사진 교체는 이 단계에서 불가.
  4. STEP4 최종 확인 — 요약 + 약관 동의 → 접수하기 → 접수확인으로 마이페이지.
  5. 저장 후 나가기 — 임시저장. 다음 접속 시 이어쓰기 여부를 묻습니다.

STEP별 핵심 동작

단계주요 동작
1회차 선택 → 시험장 선택 → TOPIK Ⅰ·Ⅱ 급수(복수) 선택 → 다음
2가입 정보 읽기 전용 표시(수정 불가, 내정보 수정 링크 제공)
3증명사진 미리보기 + 5항목 규격 체크 전부 선택 시 다음 활성
4접수 요약 확인 + 약관 동의 → 접수하기 (증명사진 없으면 제출 불가)

요소별 상세

  • 1) 단계 표시 — ①회차·급수 ②회원정보 확인 ③사진 규격 확인 ④최종 확인. 완료 단계 클릭 시 해당 단계로 이동(모바일은 활성 탭을 화면 중앙으로 스크롤).
  • 2) 저장 후 나가기 — 입력 중인 내용을 임시 저장하고 마이페이지로 나갈 수 있습니다. 저장된 내용은 30일간 보관되며, 다음에 접수 화면에 들어오면 이어서 작성할지 묻습니다.
  • 3) 회차 목록 — 접수 중·예정 회차만 카드로 노출. 카드에 제N회/제목, 상태 배지(접수 중/접수 마감/예정), 시험일, 접수 기간. 접수 중이 아닌 회차는 선택 불가. 한 번에 하나만 선택.
  • 4) 시험장 — 시험장 목록 드롭다운. 시험장이 1개면 자동 선택. 재접수이거나 같은 회차에 진행 중 급수가 있으면 기존 시험장으로 고정(변경 불가 + 안내 문구).
  • 5) 급수 선택 — TOPIK Ⅰ·Ⅱ 복수 선택. 이미 진행 중인 급수는 선택 불가 + 사유 안내(사진 심사중/수납 대기/접수 진행중). 재접수에서는 반려·취소된 급수만 다시 선택 가능.
  • 6) 응시료 표시 — 선택 회차·급수별 응시료. Ⅰ·Ⅱ 동시 선택 시 최종 확인 단계 합계에 반영.
  • 7) 회원정보 (STEP2) — 한글·영문 성명, 생년월일, 성별, 국적, 제1언어, 연락처, 이메일, 직업·응시동기·응시목적. 이 화면에서는 수정 불가 → 내정보 수정 이용.
  • 8) 사진 미리보기 (STEP3) — 가입 시 등록한 증명사진 표시. 이 단계에서 교체 불가 → 내정보 수정에서 변경.
  • 9) 규격 체크리스트 (STEP3) — 5항목: 여권용 정면 컬러(JPG)·흰색 배경 / 최근 6개월 이내 / 흑백·흐릿 아님 / 모자·선글라스·이어폰·앞머리로 얼굴 안 가림 / 정면·본인. 전부 체크해야 다음 활성.
  • 10) 최종 요약 (STEP4) — 회차·시험일, 선택 급수(2개면 "동시 접수"), 시험장, 응시자 정보, 응시료 합계. 응시료·환불 안내 함께 표시.

  3. 본부 사이트 안내에 따라 수험표 출력(보통 시험 2주 전부터)

    • 부여됐으나 공개 전: "아직 공개일 전" 경고.
    • 대상 없음/진행 중만 있음: 안내 + 접수/확인 링크.

  • 3) 안내 블록 4종 — 출력 시점(시험 2주 전 월요일 10시~), 확인 방법(로그인→마이페이지→접수현황→수험표), 시험 당일(수험표+신분증), 출력 유의사항(Chrome 권장·팝업 차단 해제).

  • 5) 기본정보 저장 — 기본정보·사진·마케팅 수신 설정을 한 번에 저장합니다.
